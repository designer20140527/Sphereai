# Sphere.ai

Sphere.ai 是一个让创作者能够通过AI创建内容，连接粉丝并通过$SPH代币赚取收益的平台。

## 技术栈

- Next.js 15.3.1
- React 19.0.0
- TailwindCSS 4

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 目录结构

- `/public` - 静态资源（图片、图标等）
- `/src/app` - Next.js 应用目录
- `/src/components` - React 组件
  - `/layout` - 布局组件（Header, Footer等）
  - `/sections` - 页面区块组件

## 部署

该项目可以部署到Vercel平台：

```bash
npm run build
```
